方法 描述 參數 備註
admin.collapseSide 側邊收縮 无 無模式限制
admin.refreshThis 刷新當前 无 無模式限制
admin.refresh 刷新指定 id: 編號 僅限 TAB 模式
admin.addTab 新增卡片 id:編號,title:標題,url:路徑 僅限 TAB 模式
admin.changeTabTitle 修改標題 id:編號,title:標題 僅限 TAB 模式
admin.closeTab 關閉卡片 id:編號 僅限 TAB 模式
admin.closeCurrentTab 關閉當前 无 僅限 TAB 模式
admin.closeAllTab 關閉所有 无 僅限 TAB 模式
admin.closeOtherTab 關閉其他 无 僅限 TAB 模式
admin.changeIframe 切換頁面 id:編號,title:標題,url:路徑 僅限 IFRAME 模式
admin.jump 相容切換 id:編號,title:標題,url:路徑 無模式限制
admin.fullScreen 全螢幕模式 无 無模式限制
