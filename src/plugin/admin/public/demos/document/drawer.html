基於 layer 的抽屜擴充

相容原版 drawer 所有參數,要使用 layer 擴充,設定 legacy 選項為 false 即可

指定容器內打開，需使用 layer 擷取層模式，並設定目標容器 style="overflow: hidden; position: relative;"
使用 url 參數指定抽屜程式碼片段位址，會使用 ajax 請求將程式碼片段拼接到抽屜中，和原來的頁面在同一個頁面上, 傳值更方便。注意片段和原頁面不能出現相同的 ID。
